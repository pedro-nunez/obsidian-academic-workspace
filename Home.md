# Home

## Daily planning

[Open today’s daily plan](obsidian://daily?vault=Workspace)

**Prepare tomorrow:** In the `Daily` folder, create a note named with tomorrow’s date in `YYYY-MM-DD` format, then run **Templates: Insert template** and select **Daily Plan**.

## Today

```tasks
not done
(scheduled today) OR (due today)
sort by urgency
group by root
```

## Needs rescheduling

```tasks
not done
scheduled before today
sort by scheduled
limit to 10 tasks
group by root
```

## Overdue deadlines

```tasks
not done
due before today
sort by due
group by root
```

## Unscheduled tasks for planning

```tasks
not done
no scheduled date
(due after today) OR (no due date)
heading does not include Waiting
path does not include Daily/
path does not include _Templates/
sort by urgency
limit to 25 tasks
group by root
```
